# Using Arrow Functions

Create a function that returns the given argument, but by using an arrow function.

An arrow function is constructed like so:

arrowFunc=(/_parameters_/)=>//code here
Examples
arrowFunc(3) ➞ 3

arrowFunc("3") ➞ "3"

arrowFunc(true) ➞ true
